Phishing Website Checker

Phishing is a type of social engineering in which an attacker sends a fraudulent message designed to trick someone into revealing sensitive information to the attacker. Phishers who create fraud websites with the goal of deluding internet users into sharing their sensitive information such as credit card details, social website accounts, password, among other data.



Tips to be safe in e-society

  • Always check whether the site you are using has a secure connection.
  • A lock icon should be present before the link of the site in the browser.
  • If not then don't enter any sensitive information.
  • Avoid using the same password in many sites and change the password in a good time gap.
  • Always check the source of the site whether it is from a trusted cohort.
  • Avoid sites with numerous ads and don’t enter any data in those redirected ad sites.
  • Ensure using the two factor authentication recommended by Google.
